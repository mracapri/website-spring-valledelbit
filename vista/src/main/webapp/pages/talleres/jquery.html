Objetivos del Curso
  • Desarrollar en el alumno la habilidad necesaria para implementar un sitio web con funcionalidades atractivas a través de jquery.
  • Que el alumno use tecnologías que simplifiquen su proceso de desarrollo de software
  • Que el alumno mejore el rendimiento de las aplicaciones en web.
  • Ampliar el panorama del alumno para utilizar recursos que le permitan resolver problemas de implementación de sitios web.
Herramientas del curso Complementos Firefox - Firebug, JSON Viewer, XML Viewer, Selenium

Extensiones Chrome - Firebug, JSON Viewer, XML Viewer
Material del Curso Durante cada día de curso se proporcionara a los alumnos un paquete comprimido con:
  • Una diapositiva la cual será ocupada durante el desarrollo de la sesión.
  • Practicas resueltas que serán ocupadas durante el desarrollo de la sesión
Temario Dia 1
  • Incorporar jQuery a proyectos de desarrollo web
  • Conceptos básicos de jQuery
Dia 2
  • CSS con jQuery
  • Aplicación de efectos con jQuery
  • Mejorando el aspecto visual de páginas web
Dia 3 y 4
  • Manejo de eventos en jQuery
  • AJAX e Interactividad con recursos web
  • Formularios
  • Manejo de datos
  • Notificaciones
Dia 5
  • Desarrollo de plugin basado en buenas practicas
  • Aplicación de pruebas unitarias al plugin
Lugar, fecha y hora El taller será en las instalaciones de la Universidad Tecnologica del Valle del Mezquital. De lunes a viernes de 18:00 a 21:00 en el laboratorio de Desarrollo web, Edificio D, Planta Alta
